Image search API

by Markus Englund

  • This API uses Bing image search
  • You can search for something by putting a keyword inside the url

Example

Go to https://url-service-yogaboll.herokuapp.com/new/http://www.aol.com/

You will receive a response that looks like this:


    {
    "short_url": "https://url-service-yogaboll.herokuapp.com/r1rNjcrVg",
    "original_url": "http://www.aol.com"
    }

The "short url" (*cough*) will redirect you to the AOL homepage!
